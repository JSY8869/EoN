# [Bubble sort](https://youtu.be/lyZQPjUT5B4)
## Bubble sort?
거품 정렬이라고도 불리는 버블 정렬은 두 인접한 원소를 검사하여 정렬(오름차순, 내림차순)하는 방법
## 문제 설명
- 주어진 값을 배열로 받아 bubble sort를 통해 
오름차순으로 정렬하시오.
- `sort` 함수는 사용 불가
- 중복된 입력은 없는것으로 가정
- python을 제외한 언어의 경우 입력되는 숫자의 개수는 6개로 통일함
## 입력 예시
1. `1 5 7 6 3 2`
2. `1 3 9 8 6 4 5 2`
## 출력 예시
1. `[1, 2, 3, 5, 6, 7]`
2. `[1, 2, 3, 4, 5, 6, 8, 9]`
